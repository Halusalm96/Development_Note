### 전처리기 컨트롤러 (UbiHost)
- 현장에서 전처리기는 IDLE 상태인데 컨트롤러는 RUN 상태여서 원인 파악
> TransferBlocked(전송 차단) Message가 오면 RUN 상태로 바뀌게 만들어져 있는 상태로 확인
> 전처리기 내부적으로 IDLE Recipe가 돌아갈 때 TransferBlocked Message를 주고 있어 상태가 다른 것으로 파악
> 전처리기 내부적으로 ILDE Recipe가 돌아갈 땐 이벤트 Message로 상태를 확인 할 수 있게 요청 완료 (이벤트 추가 - ILDE Recipe가 시작하고 끝날 때 발생)
- 추가 기능 개발 건 관련하여 시나리오 예정 확인
