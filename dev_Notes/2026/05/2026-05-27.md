- UI 통일성 확보 및 사용성 개선 요구

## 작업 내용

### LAS_UI 초안 설계 (5개)

**기존 버전 기반 3개 초안**
- [초안 1] 트리 구조: 왼쪽 탭을 트리 형태로 구성, 상위-하위 카테고리 계층 표시
- [초안 2] 트리 클릭 확장 구조: 트리 항목 클릭 시 하위 항목이 아래 방향으로 펼쳐지는 구조
- [초안 3] 중간 카테고리 위탭 고정 구조: 중간 카테고리를 상단 탭으로 고정하여 빠른 전환 지원

**신규 아이디어 2개 초안**
- [초안 4] 왼쪽 고정 탭 + 위탭 하위 카테고리 구조: 왼쪽 탭 고정, 마우스 오버 시 하위 카테고리 표시, 클릭 시 해당 페이지 이동, 같은 탭 내 페이지 이동 편의를 위해 하위 카테고리를 위탭으로 올리는 방식
- [초안 5] 위탭 고정 + 왼쪽 하위 카테고리 구조: 기존 왼쪽 고정 탭을 상단 탭으로 변경, 하위 카테고리는 위탭 아래 왼쪽 영역에 배치하여 동시 표시

**공통 개선 사항 반영 (5개 초안 전체)**
- IP/PORT 표시 추가
- 장비 추가·삭제 버튼 UI 개선
- 장비 연결 설정 페이지 수정

## 배운 점 / 느낀 점
- 탭 구조 하나를 바꾸면 하위 카테고리 접근 방식도 함께 달라져서 두 요소를 세트로 설계해야 일관성이 유지됨
- 왼쪽 고정 탭과 상단 탭은 화면 공간 활용 방식이 근본적으로 달라, 어느 쪽이 실제 사용 흐름에 맞는지 팀 피드백이 중요할 것 같음
